# GitHub Pages Yukleme Talimati

Bu paket flat hazirlandi. Tum dosyalar GitHub reposunun ana dizinine yuklenecek.

Dogru gorunum:

```text
index.html
style.css
script.js
haqqimizda.html
mehsullar.html
logistika.html
elaqe.html
anadolu-logo.jpg
murvari-logo.jpg
hero.jpg
wheat-fields.mp4
field-road.mp4
```

Bu versiyonda klasor gerekmez. Ana sayfada iki video loop icinde doner.

## Yukleme

1. Zip'i ac.
2. Icindeki tum dosyalari sec.
3. GitHub reposunda `Add file` -> `Upload files` sec.
4. Tum dosyalari ana dizine surukle.
5. `Commit changes` tikla.

## Pages

Settings -> Pages -> Deploy from a branch -> `main` -> `/root` -> Save.

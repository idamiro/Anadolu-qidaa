# GitHub Pages Yukleme Talimati

Bu paket flat hazirlandi. Yani tum dosyalar GitHub reposunun ana dizinine yuklenecek.

Bu sekilde gorunmesi dogru:

```text
index.html
style.css
script.js
haqqimizda.html
mehsullar.html
logistika.html
elaqe.html
anadolu-logo.jpg
murvari-logo.jpg
hero.jpg
wheat-fields.mp4
field-road.mp4
coverr-truck-driving-near-the-beach-8711.mp4
coverr-road-near-a-field-6275.mp4
```

Bu pakette `media` klasoru gerekmez. Video yollari ana dizine gore ayarlandi.

## Yukleme

1. Zip'i ac.
2. Icindeki tum dosyalari sec.
3. GitHub reposunda `Add file` -> `Upload files` sec.
4. Tum dosyalari ana dizine surukle.
5. `Commit changes` tikla.

## Pages

Settings -> Pages -> Deploy from a branch -> `main` -> `/root` -> Save.

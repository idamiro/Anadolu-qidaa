# GitHub Pages Yukleme Talimati

Bu klasorun icindeki dosyalari GitHub reposunun ana dizinine yukle.

Dogru gorunum:

```text
index.html
haqqimizda.html
mehsullar.html
logistika.html
elaqe.html
style.css
script.js
anadolu-logo.jpg
murvari-logo.jpg
hero.jpg
media/
assets/
```

Yanlis gorunum:

```text
anadolu-demo/
  index.html
  style.css
```

## GitHub'da Yukleme

1. Bu zip dosyasini ac.
2. Klasorun icindeki tum dosya ve klasorleri sec.
3. GitHub reposunda `Add file` -> `Upload files` sec.
4. Secilen her seyi repo ana dizinine surukle.
5. `Commit changes` tikla.

## GitHub Pages Ayari

1. Repo icinde `Settings` bolumune gir.
2. Sol menuden `Pages` ac.
3. `Source`: `Deploy from a branch`
4. `Branch`: `main`
5. `Folder`: `/root`
6. `Save`

Video icin `media` klasoru mutlaka yuklenmelidir.
